Instructions

Test 1

  1. Focus the "cardholder name" field.
  2. A credit card suggestion appears.
  3. Select the suggested credit card.
  4. All fields (WebView: except for the "CVC" field) are filled.

Test 2

Not applicable for WebView (which apparently does not offer suggestions on expiration date fields).

  1. Focus the "expiration date" field.
  2. A credit card suggestion appears.
  3. Select the suggested credit card.
  4. (Only) the cardholder name and the expiration date fields are filled.

Test 3

  1. Manually fill the form.
  2. Press the submit button.
  3. Import popup occurs.
  4. Go to chrome://settings/clearBrowserData and clear the last hour of browsing data and close the tab.
